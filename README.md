**Dashboard Course App**

**Project ini Dibuat menggunakan**

- react: ^18.2.0,
- chart.js : ^2.9.4,
- react-chartjs-2 : ^2.11.1,
- react-router-dom : ^6.3.0,
- tailwindcss: ^3.1.4
